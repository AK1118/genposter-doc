# Clip 

**Clip** 枚举用于指定组件在绘制时的裁剪（Clipping）方式。它通常与 Container、ClipRRect、Stack 等可裁剪组件搭配使用，用于控制子组件在边界之外的内容是否可见以及裁剪质量。
