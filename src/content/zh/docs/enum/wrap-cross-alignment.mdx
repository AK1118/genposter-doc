# WrapCrossAlignment

**WrapCrossAlignment** 枚举用于控制 Wrap 组件中子元素在交叉轴方向（通常是垂直方向）的对齐方式。它决定了当子元素换行时，在每一行中如何沿交叉轴排列这些元素。
