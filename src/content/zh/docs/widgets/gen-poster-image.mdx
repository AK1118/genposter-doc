# GenPosterImage

**GenPosterImage** 是一个图片组件，用于在画布中渲染图片，支持加载本地图片资源或网络图片。

使用示例

```typescript
GenPosterImage(`https://picsum.photos/1000`, {
  fit: BoxFit.cover,
  placeholderBuilder: () => Text("加载中..."),
});
```
