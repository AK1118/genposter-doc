# Opacity

**Opacity** 是 GenPoster 中用于设置子组件透明度的组件。  
通过调整透明度，你可以让子组件部分或完全透明，实现淡入淡出等视觉效果。

import { PropsTable } from '@/components/ui/props-table'


## 属性

<PropsTable
data={[
{
name: "opacity",
type: "number",
required: true,
description: "透明度，取值范围为 0.0 到 1.0，0 表示完全透明，1 表示完全不透明。",
},
{
name: "child",
type: "Widget",
required: true,
description: "需要设置透明度的子组件",
},
]}
/>

使用示例

``` typescript
// 设置子组件为 50% 透明
Opacity({
  opacity: 0.5,
  child: Container({
    width: 100,
    height: 100,
    color: Colors.red,
  }),
})
```
