# Padding

**Padding** 是 GenPoster 中用于添加内边距的组件，可以为其子组件四周添加指定的间距。

使用示例

``` typescript
// 给子组件添加上下左右各10的内边距
Padding({
  padding: EdgeInsets.all(10),
  child: Container({
    width: 100,
    height: 50,
    color: Colors.blue,
  }),
})
```

``` typescript
// 仅左右各添加16的内边距
Padding({
  padding: EdgeInsets.symmetric({ horizontal: 16 }),
  child: Text("Hello"),
})
```
