# Stack

**Stack** 是一个多子组件布局组件，用于在同一个空间内将子组件堆叠起来。每个子组件可以通过 [Positioned]() 控制其具体位置。

使用示例

```typescript
Stack({
  children: [
    Container({
      width: 200,
      height: 200,
      color: new Color(0xffcccccc),
    }),
    Positioned({
      left: 20,
      top: 20,
      child: Container({
        width: 50,
        height: 50,
        color: new Color(0xff00ff00),
      }),
    }),
    Positioned({
      right: 10,
      bottom: 10,
      child: Text("右下角"),
    }),
  ],
  alignment: Alignment.center,
  clipBehavior: "none",
});
```
