# Rotate

**Rotate** 是一个用于旋转子组件的 Widget，可用于实现任意角度的二维旋转变换。
你可以指定统一旋转角度（angle），或分别控制 x/y 方向上的旋转（angleX、angleY）。
旋转中心由 alignment 决定，默认以左上角为旋转基准点。

使用示例

```typescript
Rotate({
  child: Container({ /* ... */ }),
  angle: Math.PI / 4,
  alignment: Alignment.center,
})
```
