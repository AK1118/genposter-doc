# Scale

**Scale** 是一个用于缩放子组件的 Widget，可用于创建视觉缩放效果（例如点击放大、缩小等）。
它支持整体缩放（scale）和单轴缩放（scaleX、scaleY），默认缩放原点为组件左上角，可通过 alignment 控制缩放中心。

使用示例

```typescript

  child: Container({ /* ... */ }),
  alignment: Alignment.center,
  scale: 1.5,
})
```
