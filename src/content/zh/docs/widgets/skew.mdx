# Skew

**Skew** 是一个用于执行**斜切变换（Skew Transform）**的 Widget，适用于创建倾斜效果或模拟透视的几何变换。

通过 skewX 和 skewY 参数可分别控制在 X/Y 轴方向上的斜切角度。
单位为 弧度（radians），如：Math.PI / 6 表示 30°。

使用示例

```typescript
Skew({
  child: Container({
    width: 100,
    height: 100,
    color: new Color(0xffcc00),
  }),
  skewX: Math.PI / 12,
  skewY: Math.PI / 18,
});
```
