# Align

**Align** 是 GenPoster 中用于对子组件进行定位的组件。
它允许你在父容器的空间中对齐其子组件，例如居中、靠左、靠右、靠上等。

使用示例

``` typescript
// 将子组件放置在右下角
Align({
  alignment: Alignment.bottomRight,
  child: Container({
    width: 50,
    height: 50,
    color: Colors.blue,
  }),
})
```
或它的派生组件使用方法
``` typescript
// Center 等于 Align 居中
Center({
  child: Container({
    width: 50,
    height: 50,
    color: Colors.blue,
  }),
})
```
