# Positioned

**Positioned** 是配合 **Stack** 使用的定位组件，用于在堆叠容器中精确定位子组件的位置。

使用示例

```typescript
Stack({
  children: [
    Container({
      width: 200,
      height: 200,
      color: new Color(0xffcccccc),
    }),
    Positioned({
      left: 20,
      top: 10,
      child: Container({
        width: 50,
        height: 50,
        color: new Color(0xffff0000),
      }),
    }),
  ],
})
```
