# ClipRRect

**ClipRRect** 是一个用于裁剪子组件的容器组件，它将子组件的显示区域限制为具有圆角的矩形。与 ClipRect 不同，ClipRRect 允许通过设置圆角半径来创建更为柔和的裁剪效果，常用于实现带圆角的视觉效果。

使用示例

```typescript
ClipRRect({
  child: Container({
    width: 200,
    height: 200,
    color: new Color(0xffcccccc),
  }),
  borderRadius: BorderRadius.all(Radius.circular(20)),
  clipBehavior: "antiAlias",
});
```
