# Container

import { PropsTable } from "@/components/ui/props-table";

**Container** 是 GenPoster 中最常用的布局组件之一，用于包裹并定位子组件。它本身具备尺寸、边距、填充、背景等绘制能力，常用于快速搭建基础布局结构。

与传统前端的 div 或 Flutter 的 Container 类似，GenPoster 的 Container 完全在 Canvas 中自绘，支持灵活组合和嵌套。

示例用法

```typescript
Container({
  width: 200,
  height: 100,
  padding: EdgeInsets.all(16),
  decoration: new BoxDecoration({
    color: new Color(0xfff0f0f0),
    borderRadius: BorderRadius.circular(8),
  }),
  child: new Text("Hello GenPoster"),
});
```
