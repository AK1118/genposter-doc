# Wrap

**Wrap** 是一种自动换行的布局容器，它在主轴方向上排列子组件，如果空间不足就会自动换行到下一行（或列），非常适用于标签、图片墙等需要流式排列的场景。

与 Row 不同的是，Wrap 可以在空间不足时自动进行换行。

使用示例

```typescript
SizedBox({
  width:50,
  child:Wrap({
    children:[
      ...Array.from({ length: 10 }, (_, i) =>
        Container({
          width: 10,
          height: 10,
          color: Colors.black,
        })
      ),
    ],
    direction:Axis.horizontal
  })
});
```
