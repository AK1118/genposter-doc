# SizedBox
import { PropsTable } from '@/components/ui/props-table'


**SizedBox** 是最基础的布局构建块之一，本身不会进行任何可视化渲染。其核心作用是 在布局树中占据指定的空间，用于约束子组件或用于占位填充。通常，SizedBox 会与可渲染组件组合使用，以实现精准的尺寸控制与布局调整。
