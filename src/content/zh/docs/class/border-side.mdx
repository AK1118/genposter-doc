# BorderSide


**BorderSide** 是 Border 组件的基础构造单位，用于定义边框的样式。每个边框边都由一个 BorderSide 来描述，包含颜色、宽度等属性。


## 构造函数

``` typescript
new BorderSide({
    color: Color;
    width: number;
    style: BorderStyle;
    dashed: Iterable<number>;
})

```
