# LinearGradient

**LinearGradient** 是一种线性渐变填充方式，用于在组件的背景中呈现从一种颜色渐变到另一种颜色的视觉效果。它通常作为 BoxDecoration 的一部分进行配置。

## 构造函数

```typescript
new LinearGradient({
  colors: Color[],
  begin?: Alignment,
  end?: Alignment,
  stops?: number[]
})
```

使用示例

```typescript
Container({
  decoration: new BoxDecoration({
    gradient: new LinearGradient({
      colors: [new Color(0xFF42A5F5), new Color(0xFF1E88E5)],
      begin: Alignment.topLeft,
      end: Alignment.bottomRight
    })
  }),
  child: new Text("渐变背景")
})
```
