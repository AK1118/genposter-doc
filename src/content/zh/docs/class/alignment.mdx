# Alignment

**Alignment** 用于控制子组件在父容器中的对齐方式。它定义了一个二维坐标，x 和 y 范围为 -1.0 到 1.0，分别表示水平和垂直方向的对齐位置。

## 构造函数

``` typescript
new Alignment(x: number, y: number)
```

## 属性

import { PropsTable } from "@/components/ui/props-table";

<PropsTable
  data={[
    {
      name: "x",
      type: "number",
      default: "0",
      description: "水平对齐值，-1表示左对齐，1表示右对齐，0为居中",
    },
    {
      name: "y",
      type: "number",
      default: "0",
      description: "垂直对齐值，-1表示顶部对齐，1表示底部对齐，0为居中",
    },
  ]}
/>

使用示例

``` typescript
Alignment.topLeft         // (-1, -1)
Alignment.topCenter       // (0, -1)
Alignment.topRight        // (1, -1)
Alignment.centerLeft      // (-1, 0)
Alignment.center          // (0, 0)
Alignment.centerRight     // (1, 0)
Alignment.bottomLeft      // (-1, 1)
Alignment.bottomCenter    // (0, 1)
Alignment.bottomRight     // (1, 1)
```
