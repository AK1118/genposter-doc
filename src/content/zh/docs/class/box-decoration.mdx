# BoxDecoration

**BoxDecoration**  是 GenPoster 中用于修饰组件外观的类，它提供了颜色填充、圆角、边框等常用的视觉装饰能力。可用于 Container 等可视组件中，实现如卡片、背景块、边框线等效果。
