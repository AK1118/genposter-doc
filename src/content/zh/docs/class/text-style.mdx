# TextStyle

**TextStyle** 是一个用于配置文字样式的类，提供了字体、颜色、大小、粗细、装饰、阴影等多种属性，能够灵活地控制文字的视觉呈现效果，常与 Text 组件配合使用。

使用示例

```typescript
Text("hello world", {
  style: new TextStyle({
    color: Colors.orange,
    height: 30,
    maxLines: 4,
    textAlign: TextAlign.start,
    fontWeight: FontWeight.bold,
    fontSize: 20,
    decoration: TextDecoration.underline,
    fontFamily: "sans-serif",
    decorationColor: Colors.red,
    decorationStyle: TextDecorationStyle.dashed,
    fontStyle: FontStyle.italic,
    shadow: {
      shadowColor: Colors.black,
      shadowBlur: 10,
      shadowOffsetX: 5,
      shadowOffsetY: 10,
    },
    wordSpacing: 10,
    letterSpacing: 10,
    ellipsis: "...",
  }),
});
```
