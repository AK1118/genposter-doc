# Gradient

***abstract class***

**Gradient** 是一个抽象基类，用于构建所有渐变类（如 LinearGradient, RadialGradient 等）。该类定义了渐变颜色数组、颜色分布点以及变换矩阵。

它本身不会被直接使用，而是由具体的子类继承实现，例如 LinearGradient 会实现 createShader 方法来创建线性渐变着色器。

``` typescript
new Gradient(colors: Color[], stops: number[], transform: Matrix4)
```

使用示例
``` typescript
class MyGradient extends Gradient {
  createShader(rect: Rect): CanvasGradient {
    const gradient = ctx.createLinearGradient(rect.left, rect.top, rect.right, rect.bottom)
    this.initGradient(gradient)
    return gradient
  }
}
```
